**Запуск приложения**

1. Скачать RabbitMQ коммандой в Docker "docker run -p 15672:15672 -p 5672:5672 rabbitmq:3-management"
2. Далее собираем наш образ "docker-compose build"
3. Запускаем наш образ "docker-compose up"
